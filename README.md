❓ Problem Statement
Small to medium-scale dairy farmers often rely on manual record-keeping for cattle health, breeding cycles, and daily milk yields. This leads to:

Inaccurate data regarding milk quality and quantity.

Missed vaccination or breeding windows.

Difficulty in tracking profit/loss margins over time.

Ksheera Sagara digitizes these processes to ensure data-driven farming and improved productivity.

✨ Features
🐄 Cattle Profiles: Maintain detailed digital records for every animal.

📊 Milk Tracking: Log daily morning/evening milk production with fat content data.

📅 Health Calendar: Automated reminders for vaccinations and vet visits.

💰 Financial Reports: Automatically calculate monthly earnings based on current milk rates.

📈 Analytics Dashboard: Visual representation of farm growth and yield trends.

🛠 Tech Stack
Frontend: React Native / TypeScript (Mobile UI)

Native Modules: Kotlin (Android specific features)

Styling: CSS-in-JS

Version Control: Git & GitHub
